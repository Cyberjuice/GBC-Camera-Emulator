How to Use

  • A Button: Take a photo
  • B Button: Change filter
  • START: Download current photo
  • SELECT: Show/hide this help
  • D-Pad: Navigate UI

Keyboard Controls

  • A key: Take photo
  • B key: Change filter
  • Enter: Download photo
  • Shift: Show/hide help
  • Arrow keys: Navigate
